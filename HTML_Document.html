Game Description

This is a 2-D horizontal action game. In the game, the player can control a dragon to wipe out the enemies, which are fast moving on its way to the right hand side. By killing the enemies, player will gain scores. If the player, the dragon, is touched by the enemies, a health point will be deducted and the game is over if the health point drops to zero. There will be one stage only and player will have to face a boss at the end of the stage. The top 3 highest scores will be stored.

Tips: You can kill the enemy "mushroom" by either scratching or firing, but not stepping on it.

How to Compile

1. Unzip the file first.

2. Open Command Line.

3. Set the current directory to the unziped folder.

4. Set the PATH for jdk. (Here we use jdk1.8)

5. To compile, Type: javac src\*.java.

6. To run, Type: java -cp src Game.
